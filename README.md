# SignalClaude

Multi-session AI agent operations dashboard. Monitor, control, and communicate with Claude Code instances from your mobile device.

## Features

- **Real-time monitoring** — Watch multiple Claude Code sessions simultaneously via Socket.IO streaming
- **Mobile-first PWA** — Responsive dashboard designed for phone use
- **Permission control** — Approve or deny tool calls from your phone
- **Question & answer** — Respond to Claude's questions remotely
- **Session management** — Launch, pause, resume, and stop sessions

## Installation

### CLI (npm)

```bash
npm install -g signalclaude
```

### Desktop App

Download the latest release from the [Releases](https://github.com/signalclaude/signalclaude/releases) page.
